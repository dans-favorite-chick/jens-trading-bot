# Phoenix Bot

A locally-hosted Python + NinjaTrader 8 algo trading system for MNQ (Micro
E-mini Nasdaq-100) futures. One operator. Paper-trading phase.

**Status:** `LIVE_TRADING=False` (gated until live account ≥ $2,000 — currently
$300). Paper sim runs on 16 NT8 sub-accounts via `sim_bot`.

## Quick start

```bash
# 1. Install
pip install -r requirements.txt

# 2. Start NinjaTrader 8, load TickStreamer on the MNQM6 chart.

# 3. Start the bridge.
python bridge/bridge_server.py

# 4. Start bots.
python bots/prod_bot.py
python bots/sim_bot.py

# 5. Dashboard.
python dashboard/server.py    # then http://localhost:5000
```

See **[`docs/runbook.md`](docs/runbook.md)** for the full ops playbook.

(superseded). The only operational MDs that stay at the root are this README,
`CLAUDE.md`, and the auto-generated `SKILLS.md`.
